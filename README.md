# Blog Content Generator with Cohere API

This Streamlit app allows users to insert their Cohere API key and generate blog content based on a given topic.

## Features
- Insert Cohere API Key
- Specify blog topic and desired length
- Generate blog content using Cohere's API

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## Usage
1. Enter your Cohere API Key
2. Provide a blog topic
3. Specify the desired length of the blog
4. Click 'Generate Blog' to see the generated content

## Notes
- Ensure you have a valid Cohere API Key
- The app uses the Cohere Generate endpoint to create content
